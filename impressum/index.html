Datenschutz

Überblick

Diese Website ist ein privates Hobby-Projekt (Lernspiele für Mathematik) ohne kommerzielle Absicht. Zur Darstellung von Highscores werden freiwillig eingegebene Spielernamen und Punkte gespeichert.

Hosting — Cloudflare

Diese Website wird über Cloudflare Pages bereitgestellt. Cloudflare verarbeitet dabei technisch notwendige Daten (z.B. IP-Adressen) zur Auslieferung und zum Schutz der Website. Cloudflare ist unter dem EU-US Data Privacy Framework zertifiziert.
Weitere Informationen: cloudflare.com/privacypolicy

Highscore-Datenbank — Cloudflare D1

Wenn du einen Highscore einträgst, werden folgende Daten in einer Cloudflare D1 Datenbank gespeichert: Spielername (frei wählbar, max. 20 Zeichen), Spielname, Punktestand, Zeitstempel. Einträge werden nach 7 Tagen automatisch gelöscht (Top 50 bleiben dauerhaft erhalten). Es wird kein Account benötigt. Verwende keinen echten Namen wenn du anonym bleiben möchtest.

Standorterkennung

Zur regionalen Anzeige wird das Herkunftsland deiner Anfrage über Cloudflare ermittelt (kein separater Tracking-Dienst, nur die Cloudflare-eigene Geolokalisierung).

Schriftarten — Google Fonts

Diese Website lädt Schriftarten (Space Grotesk) von Google Fonts. Dabei wird deine IP-Adresse an Google übermittelt. Rechtsgrundlage: berechtigtes Interesse (Art. 6 Abs. 1 lit. f DSGVO).
Weitere Informationen: policies.google.com/privacy
